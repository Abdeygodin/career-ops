# Установка career-ops

## Что нужно установить один раз

### 1. Node.js 20+
Скачай и установи: https://nodejs.org (LTS версия)

### 2. ИИ-провайдер (выбери один)

Без ИИ трекер и сканирование работают. Оценка вакансий и подготовка к интервью — нет.

**Вариант А: OpenRouter — бесплатно, не нужен GPU (рекомендуется для старта)**

1. Зарегистрируйся на https://openrouter.ai
2. Создай API-ключ в разделе Keys
3. Запусти приложение → Настройки → ИИ → выбери OpenRouter → вставь ключ
4. Модель: `meta-llama/llama-3.3-70b-instruct:free`

**Вариант Б: Ollama — бесплатно, локально, нужен GPU**

Скачай и установи: https://ollama.com

```
ollama pull qwen3:14b    # GPU 8+ ГБ VRAM
ollama pull qwen3:8b     # GPU 5+ ГБ или RAM 16 ГБ (медленнее)
```

## Запуск

**Дважды кликни `start.bat`** — браузер откроется автоматически.

При первом запуске скачиваются зависимости (~100 МБ), дальше быстро.

## Первоначальная настройка

При первом открытии приложение попросит:
1. Загрузить резюме (PDF, DOCX или вставить текст)
2. Указать имя, email, целевые роли и зарплату

Всё хранится локально в папке с программой — никуда не отправляется.

## Структура папок

```
career-ops/
├── start.bat          ← запуск
├── cv.md              ← твоё резюме
├── config/profile.yml ← настройки профиля
├── data/              ← трекер вакансий
├── reports/           ← отчёты об оценке
├── output/            ← сгенерированные PDF
└── interview-prep/    ← планы подготовки к интервью
```

## Системные требования

| | Минимум | Рекомендуется |
|---|---|---|
| RAM | 8 ГБ | 16+ ГБ |
| GPU | — | Nvidia 8+ ГБ VRAM |
| Диск | 15 ГБ | 20+ ГБ |
| ОС | Windows 10/11 | Windows 11 |
