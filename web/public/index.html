📡 Telegram-каналы для сканирования

            

              Добавляй публичные каналы с вакансиями. Сканер читает последние ~20 постов через веб-превью.

              Требование: канал должен быть публичным и иметь веб-превью — открой
              t.me/s/HANDLE в браузере, там должны быть видны посты.
            


            


            

              
              
              
            

            
          


          

            

              💡 Каналы хранятся в portals.yml — можешь редактировать файл напрямую.
              Формат: telegram_channels.channels[].handle
            

          


        

      


      
      

        


          

            
🗑 Очередь (pipeline.md)

            
Помечает все ожидающие ссылки как обработанные [x] — не удаляет данные

            
            
          


          

            
📦 Трекер (applications.md)

            
Переводит вакансии в статус «Discarded» — данные остаются в файле

            

              
              
              
              
            

            

              
              
            

          


          

            
🗑 Удалить проанализированные

            
Полностью удаляет строки из трекера — используй для очистки после просмотра
